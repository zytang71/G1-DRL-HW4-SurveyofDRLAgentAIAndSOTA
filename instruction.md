# AI Harness Assignment Instruction Copy

## Objective
This assignment focuses on modern AI Harness (AI system orchestration) design.
The focus is not model training. The focus is how an AI system uses function calling,
tool integration, and workflow orchestration to execute complex tasks.

Students should demonstrate:
- LLM as a system controller
- Tool use / function calling mechanism
- Multi-step agent workflow design
- AI system architecture and data flow planning
- Basic evaluation and optimization thinking

## Requirements
Choose one AI application scenario (for example: search assistant, customer service,
data analysis agent, education assistant) and design an AI Harness system.

The design must include:
- Problem definition and usage scenario
- AI system architecture (LLM + tools + memory)
- Function calling / tool usage mechanism
- At least 3 tools (API / function) design
- Agent workflow (multi-step task execution flow)
- Evaluation method (how to measure effectiveness)
- AI orchestration (flow control and decision logic)

## Deliverables

### 1. Written Report (Required)
- Length: 2-5 pages
- Format: A4 or IEEE
- Must include:
  - Problem definition and background
  - AI Harness system design
  - Tool design (at least 3)
  - Workflow / agent flow
  - Evaluation method

### 2. Infographic (Required)
Must visualize:
- AI system architecture (LLM, tools, memory)
- Orchestration / workflow flow
- Function-calling or tool-chain flow
- Optional sequence diagram or pipeline visualization

### 3. log.md (Required)
Must record:
- Prompt/chat interaction history with AI
- System design iteration process
- Architecture adjustments and design decisions
- Issue analysis and fix process

## Evaluation Criteria (Reference)
- AI system design completeness: 35%
- Tool / orchestration design: 25%
- Workflow and logic clarity: 20%
- Infographic visual expression: 10%
- log.md process documentation quality: 10%

## Notes
- Prioritize system-design thinking, not model training or algorithm derivation.
- Clearly describe how AI performs tool use and decision-making.
- Innovative and practical application-oriented designs are encouraged.
- All designs should be logically consistent and explainable.
- Existing frameworks (for example LangChain / LangGraph) can be referenced, but are not required.
